Academic project

Predicting whether NBA players would make it into All-Star Selections using KNN classification

As part of a collaborative team, we performed in-depth exploratory data analysis (EDA) using R to predict NBA All-Star selections for the 2021-2022 season. I specialized in the data cleaning, and ggplot data visualization, and also helped work on the K-nearest neighbor (KNN) classification model. The model achieved an impressive 96.10% accuracy rate, with an optimal K value of 19 determined through cross-validation. Real-world validation confirmed our model's accuracy, aligning with the actual 2022 NBA All-Star team lineup. This is the project repository.

Technologies used include:

  • R
  • ggplot
  • Jupyter Lab
